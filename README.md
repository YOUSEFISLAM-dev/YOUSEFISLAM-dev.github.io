## Deployment

The site is hosted on GitHub Pages with an automated build system using Vite and GitHub Actions.

### Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

### Build & Deploy
```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
